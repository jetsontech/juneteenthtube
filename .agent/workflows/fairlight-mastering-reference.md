# 🎧 Fairlight Audio Mastering - Quick Reference

## Master Bus Effect Chain (add in order)

| # | Effect | Location | Key Settings |
|---|--------|----------|--------------|
| 1 | **Fairlight EQ** | EQ → Fairlight EQ | High-pass: 30Hz |
| 2 | **Multiband Comp** | Dynamics → Multiband | Preset: Broadcast |
| 3 | **Limiter** | Dynamics → Limiter | Ceiling: -1dB |

---

## Loudness Settings

| Target | Value |
|--------|-------|
| **Integrated** | -14 LUFS |
| **True Peak** | -1 dB |

---

## Quick Steps

1. **Fairlight page** → Select **Main/Master** track
2. **Add effects** (right-click effect slot)
3. **View → Loudness** to show meter
4. **Play timeline** and watch meter
5. **Adjust Main fader** until avg = -14 LUFS

---

## EQ Settings

```
High-Pass: 30Hz (removes rumble)
High Shelf: +1.5dB @ 10kHz (adds air)
```

## Limiter Settings

```
Ceiling: -1 dB
Release: 100ms
```
